M02 BASES DE DADES

INSTALAR INTELLIJ


Descargar el tar.gz de la url: aquí

Abre una terminal desde Downloads y descomprime en /opt:

sudo tar -xzf jetbrains-toolbox-1.26.5.13419.tar.gz -C /opt

Ejecuta con ./ dentro de la carpeta que acabas de descomprimir (dentro de /opt):

cd /opt/jetbrans...
sudo ./jetbrains-toolbox

Selecciona IntelliJ IDEA Community Edition para instalar:



INSTALACIÓN DE MYSQL SERVER

Sudo apt update

instalamos MySQL Server (una de las dos según si es MySQL o MariaDB):

sudo apt install mysql-server (MySQL)
sudo apt install mariadb-server (MariaDB)

iniciamos MySQL Server (una de las dos según si es MySQL o MariaDB):

sudo systemctl start mysql.service (MySQL)
sudo systemctl start mariadb.service (MariaDB)

este paso sería solamente para MariaDB. Cuando nos pida un password añadiremos daremos a enter sin introducir nada.
A continuación nos dirá si queremos poner un password a root y diremos que si Y y introduciremos el password root

sudo mysql_secure_installation

el resto de pasos simplemente le damos a todo n


entramos a la consola de MySQL (una de las dos según si es MySQL o MariaDB):

sudo mysql (MySQL)
sudo mariadb (MariaDB)

Este paso es solamente para MySQL. Modificamos el usuario root:

ALTER USER 'root'@'localhost' IDENTIFIED WITH mysql_native_password BY ‘root’;

exit


CONFIGURAR CONEXIÓN CON MYSQL

Buscamos en File > Settings > Plugins > buscamos por database y instalamos el plugin de Database Navitagor .




Reiniciamos el IDE de IntellIJ.

Nos aparece una nueva pestaña llamada DB Navigator > DB Browser > + > My SQL




Añadimos los datos de usuario y contraseña y dejamos el host a “localhost” ya que nos conectaremos al servidor que tenemos en local

y dando al botón “Test Connection” nos debe dar un Connection successful




Ahora que tenemos la base de datos enlazada, damos al botón de “Open SQL Console” y ya podemos empezar a ejecutar nuestras sentencias SQL.
